# 更新日志

本项目版本遵循语义化版本号。Alpha 版本仍可能调整界面、配置格式和内部实现。

## [1.1.0] - 2026-07-16

这是自 v1.0.3 以来的跨平台大版本更新，首次提供 Apple Silicon macOS 版本，并完善无可用系统浏览器时的备用方案。

### 新增

- 新增 Apple Silicon（arm64）macOS DMG，分项成绩导出和绩点计算器均可在 macOS 上使用。
- Edge 和 Chrome 均不可用时，可在应用内按需下载与当前 Playwright 版本匹配的备用 Chromium；安装完成后原导出任务自动继续。
- 设置页新增浏览器组件状态、版本、磁盘占用、文件位置和删除入口。
- GitHub Actions 新增 Apple Silicon 构建、Python Worker 启动、应用打包和 DMG 挂载验证。
- 正式发布流程同时生成 Windows 安装包、Windows 免安装 ZIP、Apple Silicon DMG 和统一的 SHA-256 校验文件。

### 更改

- 备用 Chromium 仅在首次需要时下载，不随安装包分发；组件保存在系统应用数据目录，升级软件时会保留。
- 浏览器组件下载支持进度显示、取消、失败重试和残留组件清理；取消时会终止相关下载进程。
- macOS 窗口改用左上角原生红黄绿控制按钮，Windows 继续使用现有窗口控制方式。
- macOS 设置、任务记录、日志、浏览器登录状态和备用浏览器统一保存在 `~/Library/Application Support/QLUToolbox`。
- README 新增 macOS 下载、升级、本地数据位置和 Apple 官方未知开发者应用打开教程。

### 修复

- 修复用户电脑未安装可用 Edge 或 Chrome 时，分项成绩查询无法正常启动的问题。
- 修复设置页打开数据位置时依赖 Windows 路径分隔符、在 macOS 上无法正确定位文件的问题。
- 修复浏览器组件下载取消或失败后，设置页不能及时显示残留文件和磁盘占用的问题。

### 兼容性与已知限制

- macOS 当前仅提供 Apple Silicon（arm64）版本，暂不支持 Intel Mac。
- macOS DMG 尚未使用 Apple 开发者证书签名和公证，首次启动可能需要按照 README 中的 Apple 官方教程手动确认“仍要打开”。
- 备用 Chromium 首次下载约 180 MiB，安装后约占用 350 MiB，需要能够访问 Playwright 浏览器下载服务。

## [1.0.3] - 2026-07-16

### 新增

- 分项成绩导出完成后新增"前往绩点计算器"引导卡片，支持一键跳转并导入刚导出的成绩。
- 关于页面 QQ 群号支持一键复制，适配 Electron、浏览器 API 和 DOM execCommand 多重剪贴板回退策略。

### 更改

- 新增统一下拉选择组件 BaseSelect，替代原生 `<select>`，支持键盘导航、级联焦点和无障碍访问。
- 设置页和分项成绩导出页全面使用 BaseSelect 组件。
- 整体字体调大（12/13/14px 体系），中文字体优先级提升，调整绿色和红色为更柔和的色调。
- 完善全局可读性样式，统一按钮、输入框和各页面排版节奏。

### 修复

- 修复绩点计算器取消勾选底部课程时界面白屏的问题。

## [1.0.2] - 2026-07-16

### 新增

- 新增绩点计算器，支持导入分项成绩 XLSX、展示全部成绩分项、自由勾选课程并实时计算总学分、总成绩点和加权平均 GPA。
- 无总评或成绩、学分无法识别的课程会保留展示并明确标记，不会自动纳入计算。
- 产品名称改为英文 `QLUToolbox.exe`，避免构建环境中文路径兼容问题。

### 修复

- 修复 GitHub Actions 发布工作流快捷方式创建失败的问题。

## [1.0.1] - 2026-07-16

### 全新界面

- 桌面 UI 从 PySide6 全面迁移到 Vue 3、TypeScript、Vite 与 Electron。
- 重做首页、工具中心、任务记录、设置、关于和分项成绩导出页面。
- 新增统一浅色/深色设计系统、响应式布局、页面转场和非阻塞通知。
- 保留原有 Python + Playwright 业务核心，通过安全 IPC 与独立 Worker 运行。

### 工程改进

- 新增长期运行的 Python Bridge，统一管理设置、任务数据库和后台任务。
- Electron 渲染进程启用上下文隔离、沙箱，并通过白名单预加载 API 使用系统能力。
- 新增 Vue/TypeScript 类型检查、前后端联合构建和 Electron 发布流程。

## [1.0.0] - 2026-07-15

首个正式发布版本，包含 v1.0.0-alpha.3 和 v1.0.0-alpha.4 中所有功能和修复。

## [1.0.0-alpha.4] - 2026-07-15

### 新增

- 任务记录页功能栏，统计展示任务总数及成功/失败/进行中数量。
- 任务记录空状态视图，无记录时显示引导提示。
- 设置页滚动区域，适配小窗口和低分辨率屏幕。
- 设置页数据管理面板，展示设置文件、任务记录、运行日志、登录状态和导出文件的完整路径，支持一键打开位置。
- 浏览器/主题设置并排网格布局，附带提示说明。
- 自定义下拉框组件（SettingsComboBox），自绘箭头图标，悬停和聚焦样式，弹出菜单圆角与选中高亮。
- 深色主题下数据路径（DataPath）等新控件的完整配色支持。

### 更改

- 任务记录表从 5 列简化为 4 列，合并"工具"和"参数"列，新增状态徽章和任务信息单元格的富界面渲染。
- 设置页保存时同步更新"导出文件"路径显示。
- README 新增"本地数据位置"章节，说明各数据目录路径及隐私提示。

### 修复

- 清除浏览器登录状态后可能残留部分 Cookie 数据的问题。
- 深色主题下状态徽章前景色不正确的问题。

### 测试

- 更新 UI 冒烟测试以覆盖新的表格列数、自定义下拉框、DataPath 字段和滚动区域。
